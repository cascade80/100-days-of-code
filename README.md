# 💯 100 Days of Code

This repository is my #100DaysOfCode journey.

For the next 100 days, I'll be documenting what I learn as I work toward becoming a better developer.

Some days will have small projects, some will have practice exercises, and others might just be experiments or notes. The goal isn't to build something huge every day—it's to keep showing up and learning.

## Structure

```

Day-02/
Day-03/
...
```

Each folder contains whatever I worked on that day.
