# Random Movie Picker

A small web app that recommends a random movie every time you click the button.

Built as Day 1 of my #100DaysOfCode challenge.

## Features

- Get a random movie recommendation
- View the movie poster, genre, release year and plot
- Uses the OMDb API to fetch movie details
- Doesn't repeat a movie until all movies have been shown

## Built With

- HTML
- CSS
- JavaScript
- OMDb API

## What I Learned

- Working with APIs using `fetch()`
- Using `async/await`
- Updating the DOM with JavaScript
- Handling API errors
- Keeping track of previously shown movies to avoid repeats

## Running the Project

1. Clone the repository.
2. Open `index.html` in your browser.
3. Click **Pick Random**.
